About RevConnexion
Using the RESTful API for Pedagogy:
  1. Create an entry in the database. Fill the form and hit Create
  2. Click on any of the entry above, edit one of the values and hit Update
  3. Click on the entry you created in (1). This will populate the form with the values of that entry. Hit Delete.
    Tip: Delete only actually required the Review ID field.
  4. Head over to the API view to see the JSON response.
  5. Change the length and offset query parameters in (4) and observe the change.
  6. Explore the API Documentation using the Swagger UI. Notice that the length and offset are optional parameters as noted in the documentation. Try to delete another entry using curl or the HTTP method as per documentation.
